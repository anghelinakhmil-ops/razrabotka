# Интернет-магазин органической косметики

Проект для GreenGlow Cosmetics — молодого бренда натуральной косметики, который за два года вырос из Instagram-магазина в полноценный бизнес с собственным производством.

## Задача

Команда GreenGlow обратилась к нам с чётким запросом: им нужен не просто интернет-магазин, а **цифровое продолжение бренда**. Сайт должен:

- Передавать философию бренда: натуральность, экологичность, осознанное потребление
- Обеспечивать удобный поиск среди 300+ товаров
- Мотивировать на повторные покупки через подписки
- Работать безупречно на мобильных — 75% аудитории приходит со смартфонов

Предыдущий сайт на конструкторе не справлялся с растущим трафиком и не давал нужной гибкости.

## Исследование

Перед началом разработки мы провели глубокий анализ:

**Аудитория:** женщины 25-40 лет, ценят натуральный состав, готовы платить за качество, активны в социальных сетях.

**Конкуренты:** проанализировали 8 ведущих магазинов органической косметики. Выявили слабые места: сложная навигация, отсутствие фильтрации по типу кожи, медленная загрузка.

**Ключевые инсайты:**
- 67% покупателей ищут продукты под конкретный тип кожи
- Средний чек растёт на 35% при наличии рекомендаций
- Подписка на регулярную доставку снижает отток на 40%

## Решение

### Дизайн

Визуальный язык строится на трёх принципах:

1. **Натуральность.** Пастельная палитра, мягкие тени, фотографии с естественным светом. Никаких кислотных акцентов — только тёплые, природные тона.

2. **Чистота.** Много воздуха, минималистичная типографика, акцент на продукте. Каждый элемент на своём месте.

3. **Доверие.** Прозрачные составы, сертификаты, реальные отзывы с фотографиями.

### Каталог и фильтрация

Реализовали умную систему поиска на базе Algolia:

- Фильтрация по типу кожи (сухая, жирная, комбинированная, чувствительная)
- Фильтрация по составу (без парабенов, без сульфатов, веган)
- Поиск с автодополнением и исправлением опечаток
- Мгновенная фильтрация без перезагрузки страницы

### Подписки

Ключевая фича — возможность оформить подписку на любой товар:

- Гибкие интервалы: каждые 2, 4 или 8 недель
- Скидка 15% на все товары по подписке
- Пауза и отмена в один клик
- Уведомления перед каждой отправкой

### Программа лояльности

Для стимулирования повторных покупок запустили бонусную программу:

- 1 балл за каждые 10 рублей покупки
- Бонусы за отзывы с фото
- Дополнительные баллы в день рождения
- Обмен баллов на скидку или подарок

### Мобильная версия

При mobile-first подходе уделили особое внимание:

- Удобная навигация одной рукой
- Быстрое добавление в корзину свайпом
- Оптимизированные изображения для мобильного трафика
- Sticky-кнопка «Купить» на карточке товара

## Технические решения

| Задача | Решение |
|--------|---------|
| Каталог 300+ товаров | Algolia Search с фасетными фильтрами |
| Подписки | Stripe Subscriptions с webhook'ами |
| Оплата | Stripe Checkout + Apple Pay / Google Pay |
| Изображения | Next.js Image + CDN, WebP, lazy loading |
| Аналитика | GA4 + Enhanced Ecommerce events |
| Уведомления | Email через Resend + Telegram Bot API |
