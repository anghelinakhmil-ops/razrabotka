# Ceramo Lab — Лаборатория материала и формы

Ceramo Lab — компания, которая работает с керамогранитом не как с товаром, а как с архитектурным материалом. Они исследуют возможности керамогранита и создают изделия под конкретные интерьерные задачи.

## Задача

Компании нужен был сайт, который передаёт их уникальный лабораторный подход к материалу. Не стандартный каталог плитки, а имиджевый сайт, показывающий два ключевых направления:

- **Изделия из керамогранита** — производство столешниц, подоконников, ступеней, панелей нестандартных размеров
- **Укладка и монтаж** — профессиональная работа с крупноформатным керамогранитом

Сайт должен вызывать ощущение премиальности, точности и экспертности.

## Решение

### Дизайн

Минималистичный стиль: много воздуха, тонкие линии, отсутствие резких смен фона. Единая светлая палитра с акцентными элементами. Типографика как основной визуальный элемент.

### Видео и медиа

Hero-секция с fullscreen видео — макро-съёмка текстуры керамогранита, медленное движение по поверхности. Видео создаёт моментальное погружение в мир материала.

### Анимации

Полноценная система scroll-анимаций на GSAP + ScrollTrigger:
- Staggered появление текста в Hero
- Параллакс изображений в секции «О нас»
- Hover-эффекты с переключением фото/видео в карточках направлений
- 4-шаговый таймлайн подхода с анимированной соединительной линией
- Zoom-эффект при наведении на проекты

### Структура

Single Page Application с якорной навигацией: 7 секций, каждая решает свою задачу — от первого впечатления до CTA.

## Технические решения

| Задача | Решение |
|--------|---------|
| Плавные анимации | GSAP + ScrollTrigger с оптимизацией для 60fps |
| Видео-фон | HTML5 video с lazy loading и poster-fallback |
| Адаптивность | CSS Modules + медиа-запросы, mobile-first |
| Производительность | Оптимизация изображений, code splitting, prefetch |
| SEO | Семантическая вёрстка, мета-теги, Open Graph |

## Результаты

Сайт передаёт премиальное позиционирование компании и чётко разделяет два направления бизнеса. Посетитель сразу понимает, что Ceramo Lab — не просто магазин плитки, а экспертная лаборатория материала.

> «Сайт точно передаёт нашу философию — мы не продаём плитку, мы работаем с материалом. Теперь клиенты приходят с правильным ожиданием.»
