# Сайт для бизнес-коуча

Проект для Анны Смирновой — опытного бизнес-коуча с 10-летним стажем.

## Задача

Клиент обратился с запросом на создание **персонального сайта**, который будет:

- Демонстрировать экспертность и опыт
- Продавать консультации и онлайн-курсы
- Автоматизировать запись на сессии
- Привлекать органический трафик через блог

## Решение

### Стратегия

Мы начали с глубокого анализа целевой аудитории и конкурентов. Выявили ключевые боли потенциальных клиентов и сформировали УТП.

### Дизайн

Выбрали минималистичный стиль с акцентом на:

- Профессиональные фотографии
- Чистую типографику
- Много воздуха между элементами

### Функционал

```
- Интеграция с Calendly
- Форма заявки с валидацией
- Блог на MDX
- Telegram-уведомления
```
