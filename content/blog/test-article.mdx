# Тестовая статья

Это тестовая статья для проверки работы **MDX** конфигурации.

## Возможности MDX

MDX позволяет использовать:

- Markdown синтаксис
- React компоненты
- JSX выражения

### Пример кода

```typescript
const greeting = "Привет, мир!";
console.log(greeting);
```

> Это цитата в MDX документе.
